## Project Title
**Shawage** — Compliance-First Private Payroll

## Short Description (150 chars)
Compliance-first private payroll: batch stealth transfers via Umbra + scoped Viewing Keys for auditors + PDF audit reports.

A DAO contributor's salary was doxxed on-chain. They quit. Shawage makes payroll invisible to everyone except authorized auditors.
